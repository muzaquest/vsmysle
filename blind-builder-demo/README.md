# Blind Builder — demo (host + players)

Это **кликабельный прототип** игры «Слепой сборщик» на собственной платформе:

- **Host (ведущий)** управляет шагами, раундами и таймером
- **Players (участники)** собирают фигуру на сетке, не видя образец
- **Architect (архитектор)** видит образец и даёт инструкции (голосом/чатом вне платформы)

Демо работает **без сервера и без базы** — синхронизация между вкладками сделана через `BroadcastChannel` (в пределах одного браузера).

## Как запустить

Из корня репозитория:

```bash
python3 -m http.server 8000
```

Откройте в браузере:

- `http://localhost:8000/blind-builder-demo/` — лендинг (выбор Host/Player)

## Как “поиграть” (самый наглядный сценарий)

1) Откройте Host:

- `http://localhost:8000/blind-builder-demo/host.html`

2) Скопируйте **Room code** (например `DEMO42`) и откройте 2–5 вкладок Player:

- `http://localhost:8000/blind-builder-demo/player.html?room=DEMO42`

3) В Host нажмите:

- **Assign Architect** (назначить архитектора)
- **Start Round** → включится таймер и появится сетка

4) Дальше игроки собирают, ведущий завершает раунд, показывает сравнение и запускает рефлексию.

## Что здесь “как в продукте”

- понятный **onboarding** по шагам
- отдельные панели **«Инструкция ведущему»** и **«Инструкция участникам»**
- роли (host / architect / builder)
- таймер и смена фаз
- фиксация результата (в демо — в пределах комнаты/вкладок)

## Ограничения демо (осознанно)

- Нет голосового/текстового чата (обычно выносится в Telegram/Zoom)
- Нет авторизации и облачного хранения
- Синхронизация работает только между вкладками одного браузера
