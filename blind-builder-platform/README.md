# Blind Builder Platform (MVP)

Это **готовая рабочая версия** игры «Слепой сборщик» на собственной платформе:

- комнаты работают между **разными устройствами** (WebSocket)
- роли: **Host / Architect / Builders**
- 2 раунда (второй — с ограничениями)
- сравнение и скоринг
- рефлексия + экспорт отчёта (Markdown)

## Запуск локально

```bash
cd blind-builder-platform
npm install
npm run dev
```

Откройте:

- `http://localhost:8787/` — лендинг
- `http://localhost:8787/host` — ведущий (создаёт комнату)

Внутри Host будет ссылка для участников вида:
`http://localhost:8787/room/XXXX`

## Как это будет жить как платформа

Сейчас в “библиотеке” одна игра. Архитектура сделана так, чтобы дальше добавлять игры как модули (новые страницы + свой state machine), а позже подключить оплату/подписку.
